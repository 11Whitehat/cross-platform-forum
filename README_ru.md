<h1 align="center">🌐 Проект Nexus Forum 💬</h1>

<p align="center" style="color: gray; font-size: 16px;">
  Современная высокопроизводительная кроссплатформенная система форума с мгновенной синхронизацией.
</p>

---

## 📱 Поддерживаемые Платформы
*   **Мобильные**: iOS 16 ~ iOS 27, Android 12 ~ Android 18
*   **ПК**: Windows 10 / 11 / 12, macOS (Apple & Intel, мин. 10.15), Linux
*   **Старые ОС**: Отдельный совместимый клиент для Windows 7 / 8 / 8.1.

---

## 🔐 Ключевые Возможности
1.  **🔄 Кроссплатформенная Синхронизация**: Мгновенная синхронизация публикаций и уведомлений между всеми устройствами.
2.  **🛡️ Двухэтапная Аутентификация**: Проверка пароля и подтверждение личности по 6-значному коду, отправляемому на e-mail.
3.  **📲 Быстрый Вход по QR-коду за 60 секунд**: Сканируйте QR-код на ПК через мобильное приложение для мгновенного входа без пароля.

---

## 🛠️ Технологический Стек
*   **Кроссплатформенный Frontend UI**: Google Flutter (Язык Dart)
*   **Облачный Сервер Аутентификации**: Node.js (Express + Безопасный токен JWT)
*   **Высокоскоростной Кэш**: Redis (Для управления жизненным циклом 5-минутных кодов подтверждения e-mail и 60-секундных динамических QR-кодов)

---

## 📂 Стандартизированная Кроссплатформенная Структура Каталогов
Этот проект использует структуру Monorepo для централизованного управления дизайн-активами, кодом клиента и кодом облачного сервера в одном репозитории:

```text
nexus_forum_project/         # 📦 Корневой Каталог Проекта
├── ui_design/               # 🎨 Независимые Активы UI Дизайна
│   ├── phone_tablet/        # 📱 Дизайн-макеты, нарезка и прототипы разметки для мобильных (Телефон/Планшет)
│   └── desktop/             # 💻 Дизайн-макеты больших экранов для десктопа (Windows/Mac/Linux)
│
├── forum_client/            # 📱💻 Кроссплатформенный Клиент Flutter (Чистая нативная компиляция, БЕЗ Web-версии)
│   ├── lib/
│   │   ├── config.dart      # ⚙️ Глобальная Конфигурация (Настройка LAN IP и порта)
│   │   ├── main.dart        # 🚀 Определение устройств, общая точка входа для регистрации и авторизации
│   │   ├── desktop_view.dart# 💻 Специально для ПК: Переключение в один клик между [Входом по QR] и [Формой пароля]
│   │   └── mobile_view.dart # 📱 Специально для Мобильных: Двухэтапный вход, [Сканер] разблокируется ТОЛЬКО после авторизации
│   └── pubspec.yaml         # 📦 Зависимости Клиента (qr_flutter, mobile_scanner и др.)
│
└── forum_server/            # 🖥️ Интегрированный Облачный Сервер Аутентификации Node.js
    ├── server.js            # 🚀 Основные Бэкенд-Сервисы (Верификация e-mail, двухэтапный перехват, 60с QR-цикл)
    └── package.json         # 📦 Зависимости Сервера (express, bcryptjs, nodemailer, jwt)
```
